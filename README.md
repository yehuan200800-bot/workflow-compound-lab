## 14 天验证标准

- 10 份有效申请
- 3 家团队完成首次安装
- 2 家团队使用真实项目累计运行至少 3 次

我们只把真实运行计为使用，不把点赞、收藏或口头好评计为验证。

## 产品阶梯（验证中）

- 入口层：首次部署 + 持续更新与答疑
- 团队层：团队配置 + 月度运行复盘
- 深度层：落地实施 + 持续托管优化

一次性部署费与月度可重复收入分开统计。

## 边界

本项目不使用既有财税产品、既有财税小红书账号或未经授权的现有渠道。演示案例会明确标注，真实案例公开前须获得许可并脱敏。
